# react boilerplate

## Setup

### 1. Get the source code

https://github.com/VetrivelKS/react-boilerplate.git

### 2. Install dependencies via yarn

```sh
$ npm install -g yarn
$ yarn install
```

### 3. Run the app

```sh
$ yarn start
```

Now you can open [http://localhost:3000](http://localhost:3000) in browser and start developing.
